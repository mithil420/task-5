# task-5

Titanic Exploratory Data Analysis (EDA)
Project Overview
======================
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns, trends, and survival factors using Python, Pandas, Matplotlib, and Seaborn.

 Files Included
 ===============
File Name	Description
EDA_Titanic.ipynb	Main Jupyter Notebook with analysis and visualizations
EDA_Titanic.pdf	PDF Report of findings and insights
train.csv	Dataset used for analysis
images/	(Optional) Saved plots & visualizations
 Key Insights
 ============================
Females had a much higher survival rate than males.
1st class passengers survived more than 2nd and 3rd class.
